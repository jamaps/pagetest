Repository to Github Pages

A simple set of instructions for future me:

  • Create new repository
  • Clone to desktop
  • Open terminal and navigate to cd /Users/.../projectfolder
  • Create a gh-pages branch by typing this in terminal:
    git checkout --orphan gh-pages
  • It should say:
    Switched to a new branch 'gh-pages'
  • In repository settings, change default branch from master to gh-pages
  • Create index.html in the project folder
  • Commit and push changes
  • Test it at jamaps.github.io/reponame
